# 概念理解面向对象编程和特点

- 面向过程：按照顺序一步一步执行。 例如，人为了抓老鼠，就买了笼子，等老鼠进来后，抓住老鼠。
- 面向对象：一个可以解决问题的对象，有解决问题的方法，那放到哪里都可以解决相同的问题，这就可以复用。人为了抓老鼠，而猫可以抓老鼠，那把猫放到哪里都可以重复做到抓老鼠。
